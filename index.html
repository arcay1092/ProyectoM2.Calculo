📘 Manual de Uso - Calculadora Científica Avanzada

Esta calculadora te permite realizar operaciones básicas y científicas...

📌 Funciones Básicas

  • AC 🧹: Limpia toda la pantalla.
  • DEL ⌫: Borra el último carácter ingresado.
  • + − × ÷ ➕➖✖️➗: Realiza operaciones aritméticas.
  • = ✅: Muestra el resultado final de la operación.

🧮 Funciones Científicas

  • π π: Inserta el valor de pi (≈ 3.1416).
  • xʸ 🔼: Eleva un número a una potencia. Ej: 2 → 🔼 → 3 = 8.
  • √ 🟰: Calcula la raíz cuadrada. Ej: 9 → √ = √(9) → = 3.
  • % 💯: Convierte un número en porcentaje. Ej: 50 → % → = 0.5.
  • log 🔟: Logaritmo en base 10. Ej: 100 → log = log(100) = 2.
  • ln ℯ: Logaritmo natural (base e). Ej: 2.718 → ln = ln(2.718) = 1.

📐 Funciones Trigonométricas

🔸 **IMPORTANTE**: Siempre ingresa primero el número, luego oprime la función.

  • sin 🟣: Seno del ángulo. Ej: 30 → sin = sin(30) → = 0.5 (en grados).
  • cos 🔵: Coseno del ángulo. Ej: 60 → cos = cos(60) → = 0.5.
  • tan 🟢: Tangente del ángulo. Ej: 45 → tan = tan(45) → = 1.
  • asin 🔺: Arco seno. Ej: 0.5 → asin = asin(0.5) → = 30.
  • acos 🔻: Arco coseno. Ej: 0.5 → acos = acos(0.5) → = 60.
  • atan 🔸: Arco tangente. Ej: 1 → atan = atan(1) → = 45.
  • DEG/RAD 🔁: Cambia entre grados y radianes. Asegúrate de elegir el modo correcto antes de usar funciones trigonométricas.

🎨 Estilo Visual

  • 🌙 / ☀️: Cambia entre modo oscuro y claro para mayor comodidad visual.

💡 Ejemplos Prácticos

  • 30 → sin → = 🔁 en DEG → Resultado: 0.5 ✅
  • 3 → √ → = → Resultado: 1.732 ✅
  • 100 → log → = → Resultado: 2 ✅
  • 2 → 🔼 → 3 → = → Resultado: 8 ✅
  • 25 → % → = → Resultado: 0.25 ✅

⚠️ Recomendaciones

  • ✔️ Asegúrate de ingresar primero el número y luego la función.
  • ✔️ Usa correctamente los paréntesis: la calculadora los coloca automáticamente.
  • ✔️ Cambia el modo DEG/RAD según lo que necesites (grados o radianes).

📈 Este manual se irá actualizando conforme se agregue nuevas funciones a la calculadora.
